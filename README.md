# LowLevelOS (LLOS)

LLOS is an W.I.P operating system writen in C

# Features:
 - Drivers:
    - PS/2 Keyboard
    - Mouse (W.I.P)
    - PC-Speaker sound
    - Filesystem (W.I.P)
    - VGA (W.I.P)
    - RTC
 - UI
    - Shell (W.I.P)
    - Desktop (W.I.P)
    - Apps (W.I.P)
 - Misc
    - Developer API (W.I.P)
